### Clone spack
```bash
cd ${HOME}
 git clone https://github.com/spack/spack.git -b 0.22.3
 cd spack
 echo ". ${HOME}/Projects/spack/share/spack/setup-env.sh" >> ~/.bashrc
```

### Clone the IOWarp Spack Repo
```bash
cd ${HOME}
git clone https://github.com/iowarp/iowarp-install.git
spack repo add iowarp-install/iowarp-spack
```

### Install IOWarp
```bash
spack install iowarp
spack load iowarp
```
